Calculate the Average of a Set of Numbers

1. Start
2. Input a list of scores
3. Initialize a variable ’sum_scores’ and set it 0
4. For each score in the list of scores
       a. Add score to ’sum_scores’
5. Initialize a variable ‘avg_score’ and set it ‘sum_scores’ divided by the number of scores in the the list
6. Output ‘avg_score’
7. End
